
# 拔河比赛

前置准备：
* 前人工作：无
* 技术栈：react + ws
* 平台选择：
  * MemFire 数据库 免费 国内 基于supabase 支持基于ws的realtime
  * Cloudflare 页面托管 后面可上 cdn 全球业务拓展预备
  * github 代码仓库


数据库：
* 有个user表
  * 主键 userId
  * username
* 历史表
  * 主键 userId
  * 输赢
  * 点击次数
  * 时间
* 房间表
  * 主键 房间id
  * 人数
  * 是否进行中

1. 玩家首先输入用户名
   1. 不存在则创建
2. 然后选择进行游戏/查看历史/旁观战局（需要房间id，未来功能）
3. 选择游戏后：
   1. 选择是开房还是加入
      1. 开房需要写人数（可选，默认两个）
   2. 选择左右
   3. 等待玩家加入
   4. 人齐后开赛
      1. 全局ws重连
      2. 假设多点击一边比另一边多点击10下获胜
   5. 比赛结果展示
   6. 回到 2
